# NightScout V4 files

Upload these files to a new GitHub repository named `nightScoutV4_files` or copy them into the root of an existing Pages repo.

## Files included
- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `apple-touch-icon.png`
- `icon-192.png`
- `icon-256.png`
- `icon-512.png`
- `icons/` duplicate icon folder

## GitHub Pages setup
1. Create a new repo named `nightScoutV4_files`
2. Upload all files from this folder to the repo root
3. In GitHub, go to **Settings > Pages**
4. Set source to **Deploy from a branch**
5. Choose **main** branch and **/(root)**
6. Save and wait for the site link

## Current V4 features
- Solo search by ZIP / city / address
- Meet in the middle
- Group center search
- Crowd, music, venue, and price filters
- Surprise venue picker
- Local rating notes saved on device
- Nearby backup venues
- Starter NJ city pack

## Important note about social scraping
Google Maps / Places alone does not scrape Instagram, venue socials, or event specials automatically.
For true live social/event scraping, add a backend or curated feed later.
This V4 build is set up so you can plug that in next.
